# Control Flow

## if

```
if condition {
    ...
}
```

## else

```
if condition {
   ...
} else {
   ...
}
```

## elif

```
if condition {
   ...
} elif condition {
   ...
} else {
   ...
}
```

## case

```
case value {
   ... # default action
} when x {
   ...
} when y {
   ...
} when z {
   ...
}
```
